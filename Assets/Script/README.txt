게임진행방식

스텟업 -> 스테이지 시작 -> 적처치 ->보스등장 -> 보스처치 ㄱ
ㄴ 코인정산 <- 사망/종료 <-   스테이지 증가 <- #유물선택#
#유물 선택은 캐릭터가 보스처치시 이번게임동안만 증가하는 스텟업입니다


게임용어
HDI : 캐릭터의 능력수치 H D I 힘 민첩 지능 3가지로 구분짓고 
증가 하는 수치에 따라 다르다.

H : 기본적인 체력 100 에 1증가에 따라 10씩 증가하고
근접 데미지가 0.2씩 올라간다.

D : 기본적인 속도 10 에 1증가에 따라 0.2씩 증가하고
공격,시전 속도가 빨라진다

I : 기본적인 마나 10 에 1증가에 따라 1씩 증가하고
마법 데미지가 1씩 올라가며 마나회복속도 는 초당 1+0.1(I) 증가한다

Coin : 영구적인 스텟을 올려주는 스텟업에 사용되는 화폐
스테이지 클리어/보스처치 (확정) ,몬스터 처리(확률) 얻을수있고 전투 종료시 코인을 정산받는다.


스텟업 : 전투시작전 플레이어가 코인을 통해 캐릭터의 스텟을 영구적으로 올릴수
있게 해준다 스텟업을 할수록 코인의 요구량은 점차 늘어난다 3렙업당 1코인을 추가로 요구한다

몬스터 : 플레이어를 공격해오는 적으로 3가지 종류가 있다
	근접 적 : 공격이 강하고 체력이 많은대신 속도가 느리다 근접공격만 할수있다
	원거리 적 : 공격이 약하고 체력이 적은대신 원거리 사격을 한다 플레이어가 오면 도망감
	이동속도 적 : 공격이 약하고 체력이 적은 대신 매우빠르다

보스 : 플레이어를 공격해오는 강한 적으로 3가지 종류가있다 다음스테이지 클리어를 위해 매 스테이지 마다 처치를 해야한다 
	전사 : 주변에 아군은 피해를 받지않는다 강한 근접공격과 높은 체력을 지니고있다
	마법사 : 플레이어를 공격하는 수많은 탄환을 발사한다 비교적 낮은 체력을 지녔다
	소환사 : 매우 빠른 해골들을 소환한다


스테이지 : 플레이어가 적들을 처리하고 보스를 처치하면 다음스테이지로 넘어갈수있다

스테이지 증가시 올라가는 수치

	스테이지넘버 : 게임이 끝날시 정산받는 코인은 스테이지 넘버수만큼 받는다
	적,보스 의 스텟 : 체력,공격력,스피드 등 증가한다
